# A New Targeted-Federated Learning Framework for Estimating HTEs

R code for the manuscript "A New Targeted-Federated Learning Framework for Estimating Heterogeneity of Treatment Effects: A Robust Framework with Applications in Aging Cohorts".

`fun_em.R` contains supporting and auxiliary functions for data generation, estimation, and results summarization.

`Transportable.R` runs the simulation study for transportable data source scenarios. The model specifications for the propensity score models and outcome regression models can be adapted to run different model specification settings.
